About Me

Hello, and welcome to my page! My name is Gary, and I am a West Coast transplant based in Providence with Bay Area roots. My passion for consulting, policy, research, and data science is rooted in my eclectic training as a nurse, medical researcher, and biostatistician. I take pride in harmonizing and leveraging my clinical, qualitative, and quantitative backgrounds to solve novel problems. You can learn more about me on this page!

Sample Projects

Infectious Disease Simulation Shiny App

This is a demonstration of a Shiny app to model infectious diseases using several models: SI, SIR, SEIR, and an adjusted model for intervention effects.

Autoencoder Demonstration

This project contains topic introduction materials and code for using autoencoders, a type of artificial neural network, for image classification.

Bayesian Analysis of Home Run Hitting

A hierarchical Bayesian model with partial pooling to predict rest-of-season home run output and an exploration of home-run rate distribution using Gaussian mixture models.

NCBI Foodborne Pathogen Detection of Listeria Isolation Source

This project aims to track and predict the source of a foodborne pathogen, Listeria, from the NCBI Pathogen Detection System using machine learning methods.

Sample Code Link for Neuroimaging Topological Data Analysis

Smooth Euler Characteristic Transform and subsequent Structured Functional Principal Component Analysis of baseline ADNI3 Tau PET scans. The files in this repo are samples for one ROI (hippocampus) and purposefully omits certain proprietary code.

Medicare Doughnut Hole Reform Paper

This writing sample, taken from a submission for a health policy course, surveys the political and economic environment/history surrounding a portion of the Bipartisan Budget Act of 2018, proposing an adjustment to the Medicare Part D coverage gap provision of the bill.
